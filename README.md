# 블로그 검색 서비스


### 목차


---
### 개발환경

- Spring Boot 2.7.9
- Java 11
- JPA
- H2
- Gradle
- Bootstrap 5
- Lombok


### 개발 내용
- Api 를 이용한 블로그 검색
  - 키워드 검색
  - Sorting : 정확도, 최신순
  - 페이징 : 최대 50페이지, 한 페이지 당 레코드 수

- 인기 검색어 표시
  - 많이 검색한 순서
  - 최대 10개


### API 명세

|테스트1|테스트2|테스트3|
